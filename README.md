# SkillScope: Data Science Job Market Analytics & Career Recommendation System

## Overview

SkillScope is an end-to-end Data Science project designed to help aspiring data professionals understand current job market trends, identify skill gaps, evaluate employability, and discover suitable career paths.

The system analyzes over 1,300 Data Science-related job postings extracted from a large LinkedIn job dataset and provides actionable insights through an interactive Streamlit dashboard.


## Problem Statement

Students often struggle to understand:

* Which skills are currently in demand
* Which Data Science role best matches their skill set
* Which skills they need to improve to become more employable

SkillScope addresses this problem by combining job market analytics with personalized career guidance.

## Dataset

Source: LinkedIn Job Postings Dataset (2023–2024)
https://www.kaggle.com/datasets/muqaddasejaz/linkedin-data-science-jobs-dataset

Original Dataset Size:

* 123,849 job postings

Filtered Dataset:

* 1,307 Data Science-related job postings

Target Roles:

* Data Scientist
* Data Analyst
* Data Engineer
* Machine Learning Engineer
* Business Intelligence Analyst
* AI Engineer


## Features

### 1. Data Science Job Market Analysis

* Analyze Data Science job postings
* Identify popular job roles
* Explore experience level requirements
* Examine work type distribution

### 2. Skill Demand Analysis

Identify the most demanded skills in the Data Science job market.

Top Skills Identified:

* SQL
* Python
* Machine Learning
* AWS
* Statistics
* Azure
* ETL
* Spark
* Tableau
* Excel

### 3. Employability Score Calculator

Users can enter their skills and compare them against current market demand.

Outputs:

* Employability Score
* Matched Skills
* Missing Skills

### 4. Career Recommendation Engine

Recommends suitable Data Science career paths based on the user's current skill set.

Supported Career Paths:

* Data Analyst
* Data Scientist
* Data Engineer
* Machine Learning Engineer
* BI Analyst
